# Pomodoro Timer
### Version 2

A simple [Pomodoro (Technique) Timer](https://en.wikipedia.org/wiki/Pomodoro_Technique). Vibe-coded.

Now with debug constants, *SECOND* and *DEBUG*.

### Requirements
- tkinter
- pygame==2.6.1

### Setup and Usage
- `pip install -r requirements.txt`
- `python pomodoro.py`
