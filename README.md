# Pomodoro Timer
### Verzion 2
A simple [Pomodoro (Technique) Timer](https://en.wikipedia.org/wiki/Pomodoro_Technique) that I vibe-coded with **Qwen** and **GPT-4o**.

Now with debug constants, *SECOND* and *DEBUG*.

### Requirements
- tkinter
- pygame==2.6.1

### Setup and Usage
- `pip install -r requirements.txt`
- `python pomodoro.py`
